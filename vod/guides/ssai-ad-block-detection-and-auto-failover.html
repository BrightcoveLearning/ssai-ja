概要

広告のフェイルオーバーを有効にすると、Brightcove Player がブラウザに広告ブロッカーが存在することを検出すると、プレーヤーは自動的にSSAI ストリームを要求します。それ以外の場合は、IMA 広告が使用されます。

SSAI 広告のフェイルオーバーには、次のものが必要です。

  • IMA3 プラグイン v3.2.0+
  • SSAI プラグイン v1.3.1+
  • ブライトコーブプレーヤーバージョン 6.8.0+

注意事項

  • 広告ブロッカーの検出は非常に信頼性がありますが、100% ではありません。
  • この機能を使用するお客様は、動的配信を使用している必要があります。
  • IMA および SSAI プラグインのみがサポートされます (FreeWheel なし)。

プレーヤーサンプル

IMA3 広告で構成されたサンプルプレーヤーは、ブラウザーで広告ブロッカーが検出されたときにサーバー側の広告に自動フェールオーバーされます。

  • 広告ブロッカーなし

    広告ブロッカーのないブラウザで以下の Codepen サンプルを実行すると、Google の DFP インラインスキップ可能な広告が表示されます。これは、IMA3、クライアント側のプレロール広告です。

  • 広告ブロッカー付き

    以下の Codepen サンプルを広告ブロッカー付きのブラウザーで実行すると、デジタルマーケティング広告の Brightcove ガイドが表示されます。これは、当社の広告設定で定義されたサーバーサイド広告（SSAI）です。

広告ブロッカーの有無にかかわらず、以下の Codepen サンプルを実行してみてください。

ペンを見るSSAI自動フェイルオーバー Brightcove Learning Services（ @ rcrooks1969） オンCodePen。

はじめに

Video Cloud からサーバーサイド広告を再生するには、以下の手順に従ってください。

  1. 広告設定を作成する
  2. ブライトコーブのプレイヤーを作成する
  3. Studio を使用した自動フェイルオーバーの実装

広告設定を作成する

広告設定では、広告コール、ビーコン、その他の設定オプションなど、SSAI 再生のさまざまな側面を定義します。広告レスポンスには、VAST、VMAP、DFP広告ルールがあります。広告設定を作成するには、次の手順に従います。

  1. Video Cloud Studio で、[ 管理者 ] メニューを展開し、[ サーバーサイド広告設定] を選択します。

  2. 広告設定の情報を追加し、[ 保存] を選択します。

詳細については、「サーバーサイド広告設定の構成」ドキュメントを参照してください。

SSAI API を使用して広告設定を作成するには、ビデオクラウド SSAI 広告設定 APIドキュメントを参照してください。

ブライトコーブプレーヤーを作成する

動画クラウドスタジオを使用して、新しいブライトコーブプレーヤーを作成します。詳細については、クイックスタートを参照してください。Player ドキュメントの作成とスタイル設定

BrightcovePlayerバージョン6.8.0以降を使用する必要があります。

Studioを使用した自動フェールオーバーの実装

自動フェールオーバー広告用にプレーヤーを設定する最も簡単な方法は、Video Cloud Studio です。広告設定とプレーヤーを作成したら、次のようにプレーヤーを自動フェールオーバー用に構成する準備が整いました。

  1. を開きますプレイヤーモジュールを作成し、広告機能を追加するプレーヤーを見つけます。
  2. プレーヤーのリンクを選択して、プレイヤーのプロパティを開きます。
  3. 左側のナビゲーションメニューで [ 広告 ] を選択します。
  4. ［クライアントサイド（IMA）を有効にする］チェックボックスをオンにします。

  5. IMA 広告タグの URL を含めます。この例では、サンプル広告タグ URL を使用します。

    プレーヤー広告のプロパティの詳細については、Players モジュールを使用したプレーヤー広告の設定を参照してください。

  6. ［サーバー側（SSAI）を有効にする］チェックボックスをオンにします。

    アカウントでSSAIが有効になっていない限り、サーバー側のオプションは表示されません。
  7. [設定の選択]ドロップダウンメニューから、このプレイヤーに関連付ける広告設定を選択します。
  8. 広告の上にオーバーレイを表示する場合は、[ 広告情報のオーバーレイを有効にする ] チェックボックスをオンにします。これには、「詳細」と広告カウントダウンオーバーレイが含まれます。

  9. クライアント側広告とサーバーサイド広告の両方を有効にすると、サーバー側広告のフェイルオーバーによる広告ブロック検出が自動的に有効になります。

  10. [ 保存] を選択します。

  11. プレーヤーを公開するには、パブリッシュと埋め込み > 変更の公開を選択します。

    プレイヤーの自動フェールオーバーを有効にするために必要なのはそれだけです。

広告で動画を再生する

動的配信で取り込んだ動画は、Video Cloud から取得した動画には、広告ブロッカーが検出されない場合にクライアント側の IMA 広告が含まれます。広告ブロッカーが検出されると、広告設定の VMAP ファイルで指定されたサーバーサイド広告が動画に含まれます。SSAI が動作するためには、ビデオにオーディオトラックが関連付けられている必要があります。
